#程式功能：
   使用兩個視窗模擬伺服端和客戶端檔案傳輸，於伺服端等待客戶端請求，當接受到客戶端請求後，檢查是否存在下載檔案，
並開始執行傳輸；如無，則傳送錯誤訊息。

#程式說明：
   在伺服端程式中，TCP listener等待客戶端請求，以執行緒方式執行，不影響伺服端視窗運作。當接受到客戶端請求後，
並執行下載服務執行緒，於執行緒當中檢查請求檔案是否存在，並執行傳輸。於執行緒當中，根據客戶端請求資訊，使用委派功能在執行緒中，
更新伺服端視窗元件訊息。
